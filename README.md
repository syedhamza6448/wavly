# Wavly

Control your PC with hand gestures — move the mouse, trigger shortcuts,
and type using a transparent on-screen keyboard.
Built with MediaPipe, PyAutoGUI and PyQt6.

## Setup
1. Clone the repo
2. Create and activate a virtual environment
3. Run `pip install -r requirements.txt`
4. Copy `.env.example` to `.env`
5. Run `python main.py`

## Tech Stack
- Python 3.10+
- OpenCV
- MediaPipe
- PyAutoGUI / pynput
- PyQt6
